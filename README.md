# FIM採点トレーナー

回復期リハビリテーション病棟のスタッフ向けに、FIM（機能的自立度評価表）の採点スキルを練習できる学習支援アプリです。

## 主な機能

- FIM採点の学習ガイド
- 練習問題によるトレーニング
- レベルアップ形式の学習システム
- スマートフォン対応（レスポンシブデザイン）
- 単一のHTMLファイルで動作（インストール不要）

## 使い方

以下のURLにアクセスするだけで利用できます。

https://suzuchan9.github.io/fim-trainer/

## ファイル構成

- `index.html` : アプリ本体（HTML/CSS/JavaScriptを1ファイルに内包）
